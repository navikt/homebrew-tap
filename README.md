# homebrew-tap

Homebrew tap for [navikt](https://github.com/navikt) CLI tools.

## Usage

```sh
brew tap navikt/tap
brew install cplt
```

## Formulae

| Formula | Description                                          |
| ------- | ---------------------------------------------------- |
| `cplt`  | macOS Seatbelt sandbox wrapper for GitHub Copilot CLI |

Formulae are automatically updated on each [cplt release](https://github.com/navikt/cplt/releases).
